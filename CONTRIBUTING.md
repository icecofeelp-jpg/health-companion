# 🤝 贡献指南

感谢您对健康管家系统的关注！欢迎提交贡献。

## 🐛 报告问题

如果您发现了 Bug 或有新功能建议，请：

1. 先搜索 [Issues](https://github.com/icecofeelp-jpg/health-companion/issues) 确认问题未被报告
2. 创建新的 Issue，使用清晰的标题和描述
3. 附上相关的截图、日志或复现步骤

## 🔧 提交代码

### 开发流程

1. **Fork 本仓库**
2. **克隆你的 Fork**
   ```bash
   git clone https://github.com/你的用户名/health-companion.git
   cd health-companion
   ```
3. **创建功能分支**
   ```bash
   git checkout -b feature/你的功能名
   ```
4. **进行修改并提交**
   ```bash
   git add .
   git commit -m "feat: 添加新功能描述"
   ```
5. **推送并创建 Pull Request**

### 提交信息规范

| 类型 | 说明 | 示例 |
|------|------|------|
| `feat` | 新功能 | `feat: 添加睡眠追踪功能` |
| `fix` | 修复 Bug | `fix: 修复定时任务不执行的问题` |
| `docs` | 文档更新 | `docs: 更新 README 使用说明` |
| `refactor` | 重构 | `refactor: 优化数据结构` |

## 📐 代码规范

- 使用中文注释解释复杂逻辑
- 保持代码简洁易懂
- 遵循 Unix 风格换行 (LF)

## 📜 许可证

通过提交代码，您同意您的贡献将基于 MIT 许可证发布。
