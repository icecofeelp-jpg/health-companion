# 🏃‍♀️ AI 健康管家系统

> 一个由多个 AI Agent 协作的健康管理系统，像朋友一样关心你的健康

## ✨ 功能特点

| Agent | 职责 | 做什么 |
|-------|------|--------|
| 💪 **健康督促员** | 每日问候 | 定时发送温暖的问候和提醒 |
| 📝 **健康记录员** | 数据记录 | 收集和记录每日健康数据 |
| 🎯 **健康顾问** | 营养建议 | 根据数据给出个性化饮食/运动建议 |
| 📊 **健康分析师** | 周报生成 | 每周生成详细的健康分析报告 |

## 🚀 快速开始

### 方法一：一键安装（推荐）

```bash
# 克隆项目
git clone https://github.com/icecofeelp-jpg/health-companion.git
cd health-companion

# 运行安装脚本
bash setup.sh
```

安装脚本会帮你：
1. 创建4个 AI Agent 角色
2. 配置健康档案
3. 设置每日提醒

### 方法二：手动安装

#### 第一步：创建 Agent 角色

```bash
hermes profile create health-recorder
hermes profile create health-analyst
hermes profile create health-motivator
hermes profile create health-consultant
```

#### 第二步：配置 SOUL.md

为每个角色创建 `SOUL.md` 文件，定义角色行为。具体内容见 [SKILL.md](./SKILL.md)。

#### 第三步：创建数据目录

```bash
mkdir -p ~/.hermes/health-records
```

#### 第四步：创建定时任务

在 Hermes 中创建以下 Cron 任务：

| 时间 | 任务 |
|------|------|
| 09:00 | 早安问候 + 体重记录 |
| 12:30 | 午餐打卡询问 |
| 15:00 | 下午喝水提醒 |
| 18:30 | 晚餐+运动询问 |
| 20:30 | 今日总结收集 |
| 22:30 | 睡前晚安 |
| 周日21:00 | 周报生成 |

## 🎯 适用人群

- ✅ 想要减脂/控制体重的职场人士
- ✅ 需要追踪健康数据的亚健康人群
- ✅ 想要养成健康习惯但缺乏自律的人
- ✅ 40+女性（特别关注代谢和骨骼健康）

## ⚠️ 注意事项

1. **首次使用需要初始化健康档案**，填写个人信息
2. **数据文件需要定期备份**，防止丢失
3. **鱼油和钙片需要另外购买**，善存复合维生素是基础款
4. **定时提醒需要配置正确的推送渠道**（飞书/微信等）
