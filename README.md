## 🚀 快速开始

### 5分钟一键安装

```bash

git clone https://github.com/icecofeelp-jpg/health-companion.git
cd health-companion

# 运行安装脚本
bash setup.sh
```

**That's it!** 安装完成后，系统会自动开始工作。

## 🔧 高级配置

### 自定义提醒时间

编辑 `~/.hermes/health-records/config.yaml`：

```yaml
reminders:
  morning: "09:00"    # 修改为你的起床时间
  lunch: "12:30"      # 修改为你的午餐时间
  afternoon: "15:00"  # 下午提醒
  evening: "18:30"    # 傍晚提醒
  summary: "20:30"    # 晚间总结
  bedtime: "22:30"    # 睡前提醒
  weekly_report: "周日21:00"  # 周报时间
```

### 添加更多补剂

```yaml
daily_supplements:
  - name: 复合维生素
    dosage: 1粒/天
    time: 早餐时
  - name: 鱼油          # 新增
    dosage: 1粒/天
    time: 随餐
  - name: 钙片          # 新增
    dosage: 500mg/天
    time: 睡前
  - name: 维生素D       # 新增
    dosage: 1000IU/天
    time: 早上
```
